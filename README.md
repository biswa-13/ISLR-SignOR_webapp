# ISLR SignOR Web App

This is a web application for Indian Sign Language Recognition (ISLR).

## Offline Mode

This web application is designed to work offline. The model is loaded locally in the browser using ONNX Runtime.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- ONNX Runtime

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
